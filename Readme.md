📌 Project Overview
---
This project is a web-based application built using HTML, CSS, and JavaScript to explore the growth patterns of aloe plants in South Africa.

The app focuses on visualizing where aloe plants (especially Aloe ferox) grow, how they develop, and how users can interact with this data through images and maps.

---
# How It Works

The user opens the website
Views aloe plant information and images
Uses the interactive map to explore where aloe plants grow in South Africa
Learns about different growth patterns based on location and environment
